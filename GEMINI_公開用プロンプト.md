## このアプリの概要（事前情報として使ってください）

**サービス名**: Gear-Loom（リポジトリ名: gear-nexus）

**説明**: 楽器・機材のレビューを共有する UGC プラットフォームです。ユーザーがレビューを投稿し、いいね・役に立った・比較リストが使えます。レビュー詳細では Amazon・楽天市場・サウンドハウス・デジマートへの検索リンクを表示しており、アフィリエイトIDを環境変数で設定できる仕様です。

**技術スタック**:
- **フロント / バック**: Next.js 15（App Router）、React 19、TypeScript
- **認証・DB・ストレージ**: Firebase（Authentication, Firestore, Storage）
- **サーバー側**: Firebase Admin SDK（API Routes で利用）
- **スタイル**: Tailwind CSS、Radix UI

**主な機能**:
- トップ: 新着レビュー・人気機材のカルーセル（横スライド）、検索、ニュース・ライブ予定・地図
- レビュー: 一覧・検索・カテゴリ、詳細（本文・画像・YouTube・いいね・役に立った・比較リスト・Xでシェア）、新規投稿・編集
- EC連携: レビュー詳細で「この機材を購入・検索する」→ Amazon / 楽天市場 / サウンドハウス / デジマートの検索リンク（Amazon・楽天はアフィリエイトタグ対応）
- ユーザー: ログイン、プロフィール・オンボーディング、マイページ、他ユーザープロフィール
- その他: 比較リスト、カスタム手帳、ブログ・フォト・イベント・ヘルプへのナビ
- 管理: 管理者用ページ（ユーザー一覧・レビュー管理など）

**環境変数（本番で必要な例）**:
- Firebase クライアント: `NEXT_PUBLIC_FIREBASE_*`（apiKey, authDomain, projectId, storageBucket 等）
- Firebase Admin: `FIREBASE_ADMIN_PROJECT_ID`, `FIREBASE_ADMIN_CLIENT_EMAIL`, `FIREBASE_ADMIN_PRIVATE_KEY`
- アフィリエイト（任意）: `NEXT_PUBLIC_AMAZON_ASSOCIATE_TAG`, `NEXT_PUBLIC_RAKUTEN_AFFILIATE_TAG`

**現在の状態**: ローカルで `npm run dev` で動作確認済み。一般公開はまだしていない。

## メモ（自分用）

- デプロイ前に `npm run build` が通ることを確認する。
